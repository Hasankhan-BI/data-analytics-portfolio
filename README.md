# Data Analytics Portfolio

A collection of Power BI dashboards built on real-world financial datasets.

---

## 1. Global Financial Markets Dashboard
- **Tools:** Power BI
- **Data:** S&P 500, NASDAQ, DAX, FTSE100 (2000–Present)
- **Features:** Price trends, volume analysis, market data table

---

## 2. Crypto Market Analytics Dashboard
- **Tools:** Power BI
- **Data:** Bitcoin, Ethereum, BNB, XRP (2015–Present)
- **Features:** Live prices, volatility tracking, coin performance table

---

## 3. Banking Transaction Analytics Dashboard
- **Tools:** Power BI
- **Data:** Bank transactions (ATM, Branch, Online channels)
- **Features:** Transaction volume over time, deposits vs withdrawals,
  top accounts, channel breakdown

Power BI · DAX · Data Modeling · Financial Analysis · Dashboard Design
